# Regressão linear múltipla e Ridge

Aqui foi feito apenas uma análise estatística de regressão linear para dados gerados aleatóriamente. O objetivo era trabalhar com as análises de resíduos e entender as métricas para avaliação de modelos. 

O programa é 100% baseado em Python.

Testes realizados:
1) QQ-plot.
2) Plot resíduos vs valores previstos.
3) Histograma dos erros com distribuição normal.

Métricas avaliadas:
1) R².
2) RMSE.
3) Viés.
